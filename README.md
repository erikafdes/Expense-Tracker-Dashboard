##  Project Scope

A fully functional, client-side **Expense Tracker Dashboard** built as a single self-contained HTML file with embedded CSS and JavaScript. The application allows users to track income and expenses, set monthly and per-category budgets, visualize spending trends, and export data — all stored locally in the browser using `localStorage`.


###  Dashboard
- Summary cards for **Total Income, Total Expenses, Net Balance, and Transaction Count**
- Month-over-month **percentage comparison** (▲/▼ indicators)
- **Bar chart** showing last 6 months of income vs. expenses
- **Donut chart** breaking down current month spending by category
- Recent 5 transactions with quick edit/delete

### Transactions
- Full transaction history with **search, filter by type, category, and month**
- Add / Edit / Delete transactions
- Each transaction has: title, amount, type (income/expense), category, date, note

### Budgets
- **Monthly spending limit** with live progress bar
- **Per-category budgets** with individual progress bars
- Color-coded warnings: green → amber (75%) → red (90%+)
- Over-budget alerts

### Analytics
- Category-wise spending bars for the current month
- Day-wise spending chart
- Top 5 biggest expenses (all time)
- Average monthly spend calculation

### Settings
- Currency symbol selector (₹, $, €, £)
- Date format selector (DD/MM/YYYY, MM/DD/YYYY, YYYY-MM-DD)
- **CSV Export** — download all transactions as a spreadsheet
- **Clear All Data** with confirmation
